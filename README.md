# Studyy Timerrrrrrrrrrr

A simple study timer built for long study sessions.

Most study timers online are based on the Pomodoro technique (25 minutes).  
This project is different.
It gives you **one fixed 8-hour session** to focus and study.
No custom timer. No distractions. Just start the timer and study.



------  Why I made this

I wanted something that encourages **long study sessions** instead of short cycles.
Many students struggle to stay focused for more periods of time.  
This timer is meant to help build habit.

 " Exactly 8 hours. No custom timer.

If you wanted a 25-minute timer… this is the wrong website. "  hahahhaa


## Features

- Fixed 8 hour countdown timer
- Start  Pause Reset controls
- Progress bar showing how much of the session is completed


## How it works

The timer starts at **08:00:00** and counts down to **00:00:00**.
